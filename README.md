# Personal Finance Tracker 💰📊

A lightweight, responsive web application built with HTML, CSS, and vanilla JavaScript to help users log income and expenses, visualize spending habits, and manage a personal budget efficiently.

---

## ✨ Features
* **Balance Dashboard:** Displays total balance, total income, and total expenses dynamically updated in real-time.
* **Transaction Logging:** Easily add transactions with a description, amount, and category (Income/Expense).
* **History Log:** A clean, scrollable history of past transactions with the ability to delete entries.
* **Persistent Storage:** Uses browser `localStorage` so data is saved and retained even after refreshing the page.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack
* **HTML5:** Structured semantic layout for form inputs and transaction tables.
* **CSS3:** Custom styling, grid/flexbox layouts, and clean animations (Optional: Tailwind / Bootstrap if used).
* **JavaScript (ES6):** State management, DOM manipulation, and data persistence using LocalStorage.
